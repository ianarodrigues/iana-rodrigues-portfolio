# 💻 Portfólio Pessoal — Iana Rodrigues

Projeto desenvolvido como atividade acadêmica da disciplina Fundamentos da Programação Web, com o objetivo de criar um portfólio pessoal utilizando HTML, CSS e JavaScript.

## 📋 Sobre o projeto

Este portfólio foi desenvolvido para apresentar informações profissionais, formação acadêmica, projetos realizados e formas de contato, reunindo em uma única página conteúdos relevantes sobre minha trajetória como desenvolvedora.

O projeto foi construído seguindo conceitos de desenvolvimento front-end, com foco em estrutura semântica, responsividade, acessibilidade e organização de código.

## 🚀 Funcionalidades

* Navegação entre seções por menu fixo
* Menu hambúrguer para dispositivos móveis
* Alternância entre tema claro e escuro
* Destaque automático do item ativo no menu durante a rolagem da página
* Layout responsivo para diferentes tamanhos de tela
* Seção de apresentação pessoal
* Seção de formação acadêmica
* Seção de portfólio com projetos desenvolvidos
* Formulário de contato
* Animações de entrada dos elementos durante a navegação

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
